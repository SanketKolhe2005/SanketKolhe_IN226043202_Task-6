📘 Assignment 6 – Function-Based Problem Solving

📌 Overview

This repository contains solutions for Task 6: Function-Based Problem Solving as part of the Internship program.

The objective of this assignment is to build practical, real-world systems using Python functions and structured programming concepts. Each problem simulates applications used in modern technology environments such as smart infrastructure, cloud computing, logistics, and event management systems.

🛠 Tools & Technologies Used

Python 3 – Core programming language

Jupyter Notebook / Google Colab – Development environment

Git – Version control system

GitHub – Code hosting and repository management

Markdown – Documentation formatting

🧠 Problems Implemented

1️⃣ Smart Parking Lot Management System

Tracks vehicle entry and exit logs

Calculates currently parked vehicles

Identifies peak parking usage

Alerts if capacity is exceeded

Real-world application: Mall parking systems, smart city infrastructure.

2️⃣ Online Food Delivery Time Estimator

Calculates delivery time based on distance

Applies traffic-based delays

Adjusts ETA according to weather conditions

Real-world application: Food delivery apps and logistics platforms.

3️⃣ Movie Theatre Seat Occupancy Analyzer

Calculates occupancy percentage

Determines if show is Housefull or Almost Full

Provides operational insights

Real-world application: Cinema ticketing systems and event platforms.

4️⃣ Cloud Server Load Classification System

Computes average CPU usage

Classifies server load as Normal, Warning, or Critical

Real-world application: Cloud monitoring dashboards and DevOps alert systems.

5️⃣ Smart Classroom Resource Usage Monitor

Tracks usage of classroom resources

Identifies overused utilities

Triggers energy alerts

Real-world application: Smart classrooms and energy optimization systems.

6️⃣ Online Event Registration Capacity Controller

Manages event registrations

Prevents overbooking

Activates waitlist when capacity is exceeded

Real-world application: Webinar platforms and conference registration systems.
